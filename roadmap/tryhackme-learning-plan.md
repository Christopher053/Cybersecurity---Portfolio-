# TryHackMe Learning Plan

## Purpose
This learning plan outlines my structured approach to developing hands-on cybersecurity skills using the TryHackMe platform. The goal is to build a strong foundation suitable for a cybersecurity internship role.

## Beginner Path (Foundation)

### 1. Introduction to Cybersecurity
- Pre Security Path
- Cybersecurity fundamentals
- Understanding threats and defenses

### 2. Linux Fundamentals
- Linux Fundamentals Part 1
- Linux Fundamentals Part 2
- Linux Fundamentals Part 3

### 3. Networking Fundamentals
- Introductory Networking
- Network Services
- How the Internet Works

---

## Security Basics

### 4. Web Fundamentals
- How Websites Work
- HTTP in Detail
- Web Application Basics

### 5. Security Tools (Beginner Level)
- Nmap (network scanning)
- Basic enumeration concepts
- Understanding ports and services

---

## Defensive Security (Blue Team Basics)

### 6. Defensive Security
- Introduction to Defensive Security
- Security Operations Center (SOC) basics
- Incident response fundamentals

---

## Learning Approach
- Complete rooms sequentially
- Take notes after each room
- Write short summaries and upload them to GitHub
- Focus on understanding concepts, not speed

## Expected Outcome
By following this plan, I aim to develop foundational cybersecurity skills, hands-on lab experience, and practical knowledge suitable for an entry-level cybersecurity internship.
